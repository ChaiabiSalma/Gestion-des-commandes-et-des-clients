# Gestion des Commandes et des Clients

## Description

Ce projet est une application de gestion des commandes pour une boutique en ligne. Elle permet à chaque client de passer plusieurs commandes, et chaque commande peut contenir plusieurs produits.

## Fonctionnalités

- *Liste des Clients* : Afficher les clients avec leurs informations (nom, adresse, email).
- *Détail des Commandes* : Afficher les commandes associées à un client, incluant la date, le montant total et les produits commandés.

## Technologies Utilisées

- *Backend* : Java EE, Spring Boot, Hibernate
- *Frontend* : Thymeleaf, Bootstrap
- *Base de données* : MySQL
